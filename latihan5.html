Belajar CSS

Apa itu CSS?

CSS atau Cascading Style Sheet adalah dokunen web yang memiliki fungsi mengatur tampilan dokumen web lain, yaitu HTML. Beberapa contoh pemanfaatan CSS pada HTML adalah untuk mengatur warna latar belakang, gaya huruf, jarak antar elemen, dan masih banyak lagi.
CSS terdiri atas kumpulan kode-kode yang keberadaannya tidak menggantikan HTHL melainkan sebagai pelengkap agar tampilan dokumen HTML menjadi lebih indah dan dinamis. Berikut ini adalah contoh dokumen CSS.

Bagaimana cara membuat CSS?

Ada cara menggunakan CSS?

  1. Inline styles : Menuliskan kode CSS langsung pada tag yang akan diubah atau diatur stylenya.
  2. Internal/embedded styles : Menuliskan kode CSS pada bagian header dokumen HTML
  3. Eksternal/linked styles : Menuliskan kode CSS pada file terpisah dan dipanggil melalui tag link dari dokumen html
